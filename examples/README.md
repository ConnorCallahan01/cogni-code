# Examples

This folder collects concrete usage examples for the current `graph-memory` plugin surface.

## Files

- [claude-code-commands.md](./claude-code-commands.md): slash command and skill examples inside Claude Code
- [mcp-tool-actions.md](./mcp-tool-actions.md): example payloads for every supported `graph_memory` action
- [skill-usage.md](./skill-usage.md): examples for the `graph-memory` skill and `/recall`
- [agent-sdk.ts](./agent-sdk.ts): local plugin usage from the Claude Agent SDK

## Scope

These examples are intentionally written against the current plugin in [`graph-memory-plugin/`](../graph-memory-plugin/), not the older root prototype.
