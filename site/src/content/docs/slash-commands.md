Installed for Claude Code, Codex CLI, and OpenCode during setup.

| Command | What it does |
|---------|-------------|
| `/memory-onboard` | First-run setup: storage, runtime, seed memory |
| `/memory-status` | Graph health snapshot |
| `/memory-search <query>` | Keyword search across all knowledge |
| `/recall <query>` | Deep graph lookup with edge traversal |
| `/memory-morning-kickoff` | Start-of-day briefing |
| `/memory-wire-project` | Inject memory context into project docs |
| `/memory-switch-harness` | Switch background pipeline worker |
| `/memory-connect-inputs` | Configure external inputs (Gmail, Calendar, Slack) |
| `/memory-input-refresh` | Refresh external input sources |
| `/refresh-skill` | Update a drifted Skillforge skill |
| `/skill-install` | Install a Skillforge skill |
| `/notion-setup` | Create Notion workspace structure |
| `/notion-sync` | Run outbound sync |
| `/notion-consolidate` | Merge batched wiki pages |

See [Skills](/skills/) for how Skillforge auto-generates additional commands.
